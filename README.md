# Luna Notes

Luna Notes is a feminine cycle planner designed for private cycle tracking, phase awareness, symptom journaling, emotional reflection, and long-term self-observation.

It helps users understand recurring patterns in energy, mood, body changes, PMS, ovulation windows, and cycle phases through a calm local-first workspace.

## Features

- Cycle tracking
- Start and finish period logging
- Phase calendar
- Ovulation window tracking
- PMS support
- Moon phase tracking
- Luna Dashboard
- Luna Life Planner
- Cycle Calendar
- Cycle Analytics
- AI Pattern Tracker
- Phase Intelligence
- Bilingual support (English / Russian)
- Local-first privacy

## Use Cases

Luna Notes can help with:

- noticing emotional changes during different phases
- tracking recurring PMS symptoms
- understanding energy patterns
- journaling thoughts by phase
- cycle-based planning
- reflective self-observation
- productivity alignment

## Privacy

All data stays inside the local vault.

No cloud storage.  
No account.  
No external tracking.

## Disclaimer

Luna Notes is built for journaling, awareness, and personal planning.  
It is not a medical tool and does not provide diagnosis, treatment, fertility prediction, or healthcare advice.
