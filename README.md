# Luna Notes

Luna Notes is a feminine cycle planner for Obsidian.

It helps you track your menstrual cycle, cycle phases, PMS, ovulation window, moon phase, wellbeing, thoughts, and personal patterns directly inside your private Obsidian vault.

## Features

- Cycle tracking
- Start and finish period logging
- Current cycle phase detection
- Ovulation window
- PMS tracking
- Moon phase tracking
- Visual cycle calendar
- Luna Dashboard
- Luna Life Planner
- Cycle Analytics
- AI Pattern Tracker
- Phase Intelligence
- Russian and English language support
- Local-first data storage

## Privacy

All data is stored locally inside your Obsidian vault.

No cloud storage.  
No account required.  
No tracking.

## Disclaimer

Luna Notes is for self-observation, journaling, and planning. It is not a medical app and does not provide diagnosis, treatment, contraception advice, or medical recommendations.
