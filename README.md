# 🌙 Luna Notes

A feminine cycle planner for Obsidian designed to help you track your body, energy, symptoms, patterns, and emotional changes — all in one private local-first workspace.

---

## ✨ Features

- 🌸 Cycle tracking
- 🩸 Start / Finish period logging
- 📅 Current cycle phase detection
- 🌙 Moon phase tracking
- ⚡ PMS logic
- 🥚 Ovulation window
- 📊 Luna Dashboard
- 📝 Luna Life Planner
- 📆 Cycle Calendar
- 📈 Cycle Analytics
- 🧠 AI Pattern Tracker
- 🔮 Phase Intelligence
- 🌍 Russian / English support
- 🔒 Local-first storage (your data stays yours)

---

## 💜 Why Luna Notes?

Luna Notes was built for people who want a more mindful and private way to understand their body inside Obsidian.

Instead of using traditional health apps, Luna Notes helps you combine:

- Cycle awareness
- Productivity planning
- Symptom tracking
- Mood changes
- Energy patterns
- PMS understanding
- Ovulation insights
- Long-term analytics

Everything stays inside your vault.

## 🚀 Installation

### Community Plugins (after approval)
1. Open Obsidian
2. Go to **Settings → Community Plugins**
3. Search for **Luna Notes**
4. Install and enable

### Manual Installation
1. Download latest release from GitHub
2. Open your vault:
   `.obsidian/plugins/luna-notes/`
3. Copy:
   - `main.js`
   - `manifest.json`
   - `styles.css`
4. Restart Obsidian
5. Enable Luna Notes in Community Plugins
